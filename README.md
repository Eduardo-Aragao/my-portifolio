# 🚀 Projeto de Estudo: Desenvolvimento Front-end

Este repositório foi criado para aprimorar meus conhecimentos em tecnologias modernas de desenvolvimento web. O foco principal aqui é aplicar boas práticas de componentização, tipagem estática e performance, indo além do "boilerplate" inicial.

## 🛠️ Tecnologias Utilizadas
Este projeto utiliza o que há de mais atual no ecossistema:
- **Vite:** Build tool extremamente rápida para o desenvolvimento.
- **TypeScript:** Garantindo segurança com tipagem e melhor manutenção do código.
